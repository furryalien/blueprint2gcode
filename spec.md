Create a new application. The purpose of this application is to convert an image to gcode that can be plotted. 

this applicaiton is optimized for blueprint style images that are jpg or png files

It prefers to use 1.0 mm pens.
It prefers to using smooth single lines.
It prefers to minimize the pen movement and choose an efficient path for the page.
It prefers to create an output that will fit an A4 sheet of paper.
It prefers to scale input images to the output size
